> Automatically capture, narrate, and export interactive product demos as videos and PDFs — powered by generative AI.

## The Challenge

SaaS companies need product demos fast. Manually recording screens, editing footage, and writing scripts doesn't scale. The platform needed to automatically capture dynamic web interfaces and transform them into polished demo assets with AI-generated narration.

## Key Engineering Decisions

### Hybrid AI with BYOK

`pgvector` semantic search combined with multi-LLM generation (OpenAI & Gemini). Enterprise clients bring their own API keys — a token ledger tracks all inferences per organization with role-based access control.

### Headless Media Pipeline

Background workers convert live React/DOM scenes into video and PDF assets using **Puppeteer** for capture and **FFmpeg** for encoding.

```typescript
export async function convertDOMToMedia(session: DOMSession) {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  await page.setContent(session.htmlSnapshot);

  const recordingStream = await page.createPDFStream();
  const converter = spawn("ffmpeg", ["-i", "pipe:0", "-c:v", "libx264", "/outputs/demo.mp4"]);
  recordingStream.pipe(converter.stdin);

  await s3.upload(converter.stdout);
  return { status: "SUCCESS" };
}
```

### Usage-Based Billing

The token ledger bills AI credit consumption per organization, preventing platform-wide rate limits from affecting individual tenants.

## Impact

- **DOM to video in seconds** — Decoupled Puppeteer + FFmpeg workers handle heavy rendering without blocking the API
- **Enterprise-safe AI** — BYOK integration with per-org usage tracking and RBAC
- **Fully automated demos** — From DOM capture to narrated video, no manual editing required
