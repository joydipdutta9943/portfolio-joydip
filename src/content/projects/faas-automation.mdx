> Automating the deeply complex lifecycle of alternative investment funds via headless reporting engines and mathematically perfect relational flow structures.

## Key Architecture & Engineering

### Automated Document Pipeline

Replaced days of manual calculations by building an in-memory generation pipeline using headless **Puppeteer**. It hydrates HTML templates with complex financial data to render thousands of personalized PDF notices, streaming them directly to **AWS S3** to completely bypass slow disk I/O limitations.

```typescript
// The PDF Clustered Generation logic
export async function streamNoticesToS3(investors: Investor[]) {
  const browser = await puppeteer.launch({ headless: "new" });
  const promises = investors.map(async (investor) => {
    const page = await browser.newPage();
    await page.setContent(buildLegalTemplate(investor));

    // In-memory buffer bypasses rigid standard I/O
    const pdfBuffer = await page.pdf();
    await uploadToAWS(`funds/drawdown/${investor.id}.pdf`, pdfBuffer);

    await page.close();
  });

  await Promise.all(promises);
  await browser.close();
}
```

### Complex Financial Modeling

Designed a rigorous relational ledger schema using **Drizzle ORM** to accurately track multi-layered capital flows (`Fund` → `Scheme` → `Investor` → `Commitment` → `Drawdown`) down to the exact cent.

### Strict Multi-Tenancy

Enforced absolute operational data isolation via a custom `ContextService` middleware that automatically scopes every database query to the specific organization's ID, fully preventing cross-tenant data bleed.
